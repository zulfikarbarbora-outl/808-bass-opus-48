# 808-bass-opus-48

Sustained 808 / sub bass one-shots at multiple pitches for hip-hop/trap basslines. Mix of true 808 sustained tones (James_KuKu '808 C note' pack 40715, CC0) and chromatic sustained analog synth bass from MFB Kraftzwerg 'Boo Bass' pack 17440 by modularsamples (CC0).

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Library**: 808 / Sub Bass One-Shots
- **Files**: 12 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **License**: CC0-1.0
- **Source**: https://freesound.org/people/James_KuKu/packs/40715/
- **Branch**: `main`
